# Part 2- Playing with Numbers.

```py
int_num = 10
float_num = 20.0

print(int_num)
print(float_num)

b = 20

print("*******************")

add = a + b
print(add)

sub = b - a
print(sub)

multi = a * b
print(multi)

div_mychoice = b / a
print(div_mychoice)
```

# Part 3- Get to know python Boolean .

```py
a = True
b = False

print(a)
print(b)

print("**************************")
print(bool(0))
print(bool(1))
print(bool(2))

c = ""
print(bool(c))

c = "Some Value"
print(bool(c))
```
